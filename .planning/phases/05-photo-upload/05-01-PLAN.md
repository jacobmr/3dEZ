# Phase 5, Plan 1: Photo Upload Infrastructure

<objective>
Create backend photo upload endpoint with Photo DB model, and frontend PhotoUpload component with mobile camera capture, gallery selection, and client-side image compression. Photos stored on disk with metadata in SQLite.
</objective>

<execution_context>

- Existing backend: FastAPI async, SQLAlchemy async with aiosqlite, UUID4 PKs
- Existing frontend: Next.js 16.x, Tailwind v4, TypeScript strict
- Message model: `content: Text`, `tool_use: JSON` — needs `photo_ids: JSON` for image refs
- MessageInput.tsx is text-only textarea — needs photo attach button
- API client uses `X-Session-ID` header for session identification
- Docker volume at `/app/data` for persistence
  </execution_context>

<context>
Key files:
- backend/src/app/db/models.py — SQLAlchemy models (Session, Conversation, Message, Design)

- frontend/src/components/chat/ChatPanel.tsx — chat message list + input
- frontend/src/lib/api.ts — API client functions
- frontend/src/hooks/useConversation.ts — conversation state management

Design constraints:

- Max photo size: 5MB after compression (Claude Vision limit is 20MB but we want fast uploads)
- Client-side resize to max 1568px on longest edge (Claude Vision optimal resolution)
- JPEG compression at quality 0.85 for photos
- Store originals on disk at `data/photos/{session_id}/{uuid}.jpg`
- Photo model links to conversation (not message) — photo can be referenced across messages
  </context>

## Task 1: Backend photo upload endpoint and Photo model

type: auto

**Add Photo model to `backend/src/app/db/models.py`:**

```python
class Photo(Base):
    """Uploaded photo for vision analysis."""
    __tablename__ = "photos"

    id: Mapped[str] = mapped_column(String(36), primary_key=True, default=_new_uuid)
    session_id: Mapped[str] = mapped_column(ForeignKey("sessions.id", ondelete="CASCADE"))
    conversation_id: Mapped[str] = mapped_column(ForeignKey("conversations.id", ondelete="CASCADE"))
    filename: Mapped[str] = mapped_column(String(255))
    content_type: Mapped[str] = mapped_column(String(50), default="image/jpeg")
    file_path: Mapped[str] = mapped_column(String(500))  # relative to data dir
    file_size: Mapped[int] = mapped_column(Integer)
    created_at: Mapped[datetime] = mapped_column(DateTime(timezone=True), server_default=func.now())

    session: Mapped[Session] = relationship()
    conversation: Mapped[Conversation] = relationship()
```

```typescript
export async function uploadPhoto(
  conversationId: string,
  file: File,
): Promise<{ id: string; filename: string }> {
  const formData = new FormData();
  formData.append("file", file);
  const res = await fetch(`/api/conversations/${conversationId}/photos`, {
    method: "POST",
    headers: { "X-Session-ID": getSessionId() },
    // No Content-Type — browser sets multipart boundary
    body: formData,
  });
  return json<{ id: string; filename: string }>(res);
}
```
