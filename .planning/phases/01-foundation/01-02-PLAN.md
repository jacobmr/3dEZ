- `src/app/globals.css` — Tailwind directives plus `html, body { height: 100%; }` for full-height mobile layout.
- `src/app/page.tsx` — Simple landing page with "3dEZ" heading and a placeholder "Start Designing" button. Mobile-first centered layout.

Use App Router (not Pages Router). TypeScript strict mode. Do NOT set up Three.js scene yet — just install the deps.
