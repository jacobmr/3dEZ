Create frontend/src/lib/session.ts:
- getSessionId(): Get or create session UUID in localStorage
- Returns string UUID

Create frontend/src/lib/api.ts:
- API helper functions with X-Session-ID header:
  - createConversation(message: string): POST /api/conversations
  - listConversations(): GET /api/conversations
  - getConversation(id: string): GET /api/conversations/{id}
  - deleteConversation(id: string): DELETE /api/conversations/{id}
  - listDesigns(): GET /api/designs
  - streamMessage(conversationId: string, message: string): returns EventSource-like async iterator that parses SSE events from POST /api/conversations/{id}/message
  - streamRevision(conversationId: string, message: string): same for /revise
- For SSE POST requests: use fetch() with ReadableStream, parse SSE format manually (EventSource only supports GET). Create a parseSSE(response: Response) async generator.

Create frontend/src/hooks/useConversation.ts:
- React hook managing conversation state
- State: messages[], isStreaming, currentDesign, error
- Methods: sendMessage(text), reviseDesign(text), loadConversation(id)
- Handles SSE stream parsing, appends text chunks to streaming message, processes tool events

Create frontend/src/components/chat/:

MessageList.tsx: Scrollable message list
- User messages: right-aligned, blue/indigo bg
- Assistant messages: left-aligned, dark bg
- Auto-scroll to bottom on new messages
- Show typing indicator during streaming

StreamingMessage.tsx: Renders in-progress assistant message
- Displays text as it streams in
- Shows "Thinking..." before first chunk
- On parameters_extracted event: show a "Design ready!" card with category and key dimensions

MessageInput.tsx: Input bar at bottom
- Text input + send button
- Disabled during streaming
- Enter to send, Shift+Enter for newline
- Placeholder text: "Describe what you want to create..."

ChatPanel.tsx: Composes the above into the conversation panel
- If no active conversation: show welcome message + prompt
- If conversation active: MessageList + MessageInput
- New conversation button in header area
