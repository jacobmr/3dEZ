<task type="auto">
  <name>Task 1: Define design parameter Pydantic models and Claude tool schemas</name>
  <files>backend/src/app/models/designs.py, backend/src/app/models/tools.py</files>
  <action>
Create backend/src/app/models/designs.py with Pydantic v2 models:

BaseDesignParams: category (Literal), units (Literal["mm","in"] = "mm"), notes (str = "")

MountingBracketParams(BaseDesignParams):

- category: Literal["mounting_bracket"]
- width: float (mm, the mounting surface width)
- height: float (mm, vertical dimension)
- depth: float (mm, how far bracket extends from wall)
- thickness: float = 3.0 (mm, wall thickness)
- hole_diameter: float = 4.5 (mm, mounting hole size — M4 default)
- hole_count: int = 2
- lip_height: float = 5.0 (mm, retaining lip on shelf edge)

EnclosureParams(BaseDesignParams):

- category: Literal["enclosure"]
- inner_width: float
- inner_height: float
- inner_depth: float
- wall_thickness: float = 2.0
- lid_type: Literal["snap", "slide", "screw", "none"] = "snap"
- ventilation_slots: bool = False
- cable_hole_diameter: float = 0.0 (0 = no hole)
- corner_radius: float = 2.0

OrganizerParams(BaseDesignParams):

- category: Literal["organizer"]
- width: float
- height: float
- depth: float
- compartments_x: int = 1 (grid columns)
- compartments_y: int = 1 (grid rows)
- wall_thickness: float = 1.5
- has_labels: bool = False
- stackable: bool = False

DesignParamsUnion = MountingBracketParams | EnclosureParams | OrganizerParams (discriminated union on category field)

Create backend/src/app/models/tools.py:
Define Claude tool schemas as Python dicts (matching Anthropic tool format) for two tools:

1. "extract_design_parameters" — called when Claude has gathered enough info to produce a full parameter set. Input schema mirrors DesignParamsUnion as JSON Schema.
2. "request_clarification" — called when Claude needs specific information from the user. Input: { question: str, options: list[str] | null, parameter_name: str }

These are the tools passed to client.messages.create(tools=...). Use the Anthropic tool format: { name, description, input_schema }.

Validate tool schemas match Pydantic models by generating JSON Schema from models: MountingBracketParams.model_json_schema().
