# Plan: 09-03 Iteration UX Polish

## Objective

Polish the conversational iteration experience with quick-edit suggestions and smoother transitions.

## Context

After 09-01 and 09-02, the core iteration loop works. This plan adds UX polish: suggested modifications, quick parameter tweaks, and visual refinements.

## Tasks

- [x] **Task 1: Suggested modifications**
      After design generation, have Claude suggest 2-3 common modifications as clickable chips below the preview (e.g., "Make it wider", "Add ventilation holes", "Round the corners"). Backend: add `suggest_modifications` field to `parameters_extracted` event. Frontend: render as styled chips that auto-send as messages when clicked.

- [x] **Task 2: Quick parameter nudge**
      Add +/- buttons on dimension values in the ParameterDiff component for quick numeric tweaks (e.g., width ±5mm). Clicking triggers a revision with the adjusted parameter. Simple, direct parameter editing without conversational round-trip.

- [x] **Task 3: Loading and transition states**
      Add skeleton loading state while STL regenerates on revision. Show "Regenerating..." overlay on preview during generation. Smooth opacity transition when new STL loads. Disable send button during generation to prevent concurrent modifications.

## Notes

- Suggested modifications come from Claude's context awareness
- Quick nudge is a shortcut — still creates proper design versions
- Focus on perceived speed and responsiveness
