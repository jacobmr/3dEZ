## Task 1: Create StlMesh and StlViewer components

Create the core R3F rendering components.

**Files to create:**

1. `frontend/src/components/preview/StlMesh.tsx`
   - Props: `stlBytes: ArrayBuffer`
   - useMemo to parse STL: `new STLLoader().parse(stlBytes)`
   - computeBoundingBox(), center(), computeVertexNormals()
   - Render `<mesh>` with `<meshStandardMaterial>` (neutral gray-blue, metalness 0.3, roughness 0.6)
   - Dispose geometry on unmount via useEffect cleanup

2. `frontend/src/components/preview/StlViewer.tsx`
   - Props: `stlBytes: ArrayBuffer`
   - R3F scene contents (goes inside Canvas):
     - `<StlMesh stlBytes={stlBytes} />`
     - `<ambientLight intensity={0.4} />`
     - `<directionalLight position={[5, 5, 5]} intensity={0.8} />`
     - `<directionalLight position={[-3, -3, 2]} intensity={0.3} />`
     - drei `<OrbitControls makeDefault enableDamping dampingFactor={0.1} />`
     - drei `<ContactShadows>` on ground plane (optional, subtle)
   - Export as default

**Verification:**

- `cd frontend && npx tsc --noEmit` passes
