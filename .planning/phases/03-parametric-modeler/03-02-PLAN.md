  <action>
Create `generate_mounting_bracket(**params) -> Part` following the research code examples:

Parameters (from MountingBracketParams, excluding category/units/notes):

- width, height, depth, thickness, hole_diameter, hole_count, lip_height

Geometry:

1. Base plate: `Box(width, depth, thickness)` flat on XY plane
2. Vertical wall: Position at back edge, `Box(width, thickness, height)` — use `Pos()` to place
3. Fuse base + wall to get L-bracket shape
4. If lip_height > 0: Add retaining lip at top of wall, projecting forward `thickness` amount
5. If hole_count > 0 and hole_diameter > 0: Subtract mounting holes in base plate using `BuildPart` + `GridLocations` + `Cylinder`. Space holes evenly across width.
6. Apply fillets on inner L-corner edges LAST (after all booleans). Use try/except for fillet — if it fails, skip fillets and log warning (OCCT can fail on certain geometries).
7. Validate: `part.volume > 0`, bounding box approximately matches (width, depth, height)

Return the Part object. Do NOT export STL here — engine handles that.
</action>

  <action>
**Enclosure** (`generate_enclosure(**params) -> Part`):
Parameters: inner_width, inner_height, inner_depth, wall_thickness, lid_type, ventilation_slots, cable_hole_diameter, corner_radius

Geometry:

1. Outer box: dimensions = inner + 2\*wall_thickness on each axis
2. Inner cutout: `Pos(0, 0, wall_thickness) * Box(inner_width, inner_depth, inner_height + wall_thickness)` — extends above to open top
3. Subtract inner from outer for shell
4. If cable_hole_diameter > 0: Cut a cylinder through one side wall
5. If corner_radius > 0: Apply fillet on vertical outer edges LAST. Try/except for fillet failures.
6. Skip lid_type geometry for now (just the body) — lid is a future enhancement
7. Skip ventilation_slots for now — future enhancement

**Organizer** (`generate_organizer(**params) -> Part`):
Parameters: width, height, depth, compartments_x, compartments_y, wall_thickness, has_labels, stackable

Geometry:

1. Outer box: `Box(width, depth, height)`
2. Inner cutout: open top, wall_thickness offset on bottom and sides
3. Add X dividers: `compartments_x - 1` vertical walls evenly spaced along width
4. Add Y dividers: `compartments_y - 1` vertical walls evenly spaced along depth
5. Fuse all dividers with shell
6. Skip has_labels and stackable features for now — future enhancement

**Register all templates** in `templates/__init__.py`:

- Import all three template functions
- Create `TEMPLATE_REGISTRY: dict[str, Callable]` mapping category strings to functions
- In modeler `__init__.py`, auto-register all templates from registry when ModelEngine is instantiated
